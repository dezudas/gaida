

> A minimalist CSS framework on SCSS version, forked from Milligram.


## Download

**Install with Bower**

```sh
$ bower install gaida
```

**Install with npm**

```sh
$ npm install gaida
```

**Install with Yarn**

```sh
$ yarn add gaida
```
